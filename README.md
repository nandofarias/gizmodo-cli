# Gizmodo Reader CLI
Read all informations and comments from gizmodo.uol.com.br without leaving your terminal

## Instalation
```bash
$ npm install -g gizmodo-cli
```

## Usage
First you run the cli
```bash
$ gizmodo-cli
```
or
```bash
$ gizmodo-cli list
```

Then you choose one of the news from the list
